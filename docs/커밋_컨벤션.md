| Tag Name | Description                                            | ex)                           |
| -------- | ------------------------------------------------------ |-------------------------------|
| feat     | 새로운 기능을 추가                                     | `[feat] #10 로그인 api 작성`       |
| fix      | 코드 버그 수정                                         | `[fix] # 11 로그인 enum type 수정` |
| remove   | 코드나 파일을 삭제하는 작업만 수행한 경우              | `[remove] #12 불필요한 import 제거` |
| refactor | 로직 변경없이 코드를 개선하는 리팩토링                 | `[refactor] #13 로그인 코드 개선`    |
| chore    | 의존성 추가, yml 추가 및 수정, 패키지 구성 등 업데이트 | `[chore] # 14 lombok 의존성 추가`  |
| test     | 테스트 코드, 리펙토링 테스트 코드 추가                 | `[test] #15 로그인 테스트 코드 작성`    |
| docs     | readme등의 문서 수정                                   | `[docs] #16 README 수정`        |
| deploy   | 배포 관련                                              | `[deploy] main 브랜치 병합`        |
| hotfix   | 긴급 버그 수정                                         |                               |
